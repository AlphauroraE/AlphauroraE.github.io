# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository overview

This is a GitHub Pages user site (repo name `AlphauroraE.github.io`), served directly from `index.html` at the repository root with no build step, bundler, or dependencies. GitHub Pages publishes whatever is on the default branch as-is.

## Development

There is no build, lint, or test tooling in this repository — it is plain static HTML. To preview changes, open `index.html` directly in a browser or serve the directory locally, e.g.:

```
python3 -m http.server
```

Changes pushed to the default branch on GitHub go live automatically via GitHub Pages.
